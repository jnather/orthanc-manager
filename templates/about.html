About This Project

This Orthanc Admin interface was developed to provide an intuitive and easy-to-use platform for managing DICOM modalities and monitoring Orthanc server status. It offers a clean UI and efficient tools for healthcare professionals and developers alike.

Author

Developed by Julio Cesar Nather Junior in 2024. Julio is a radiologist and developer with experience in medical imaging and AI technologies.

Purpose

The purpose of this project is to streamline the management of DICOM modalities and provide a user-friendly interface for healthcare facilities using the Orthanc DICOM server. It aims to enhance productivity and efficiency in medical imaging workflows.

If you have any questions or feedback, please feel free to reach out to Julio at julio.nather@hcrp.usp.br.
